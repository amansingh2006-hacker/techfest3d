# techfest3d
Design and develop a 3D interactive website for Techfest. Think 3D scroll animations, interactive objects and dynamic transitions. You have full creative freedom on the concept and visuals.

## ✨ Features

* **3D Reality Warp:** A fully functional 3D canvas that acts as a continuous background layer.
* **Scroll-Linked Camera Animations:** As the user scrolls, the virtual camera flies through the 3D space, transitioning into different festival zones.
* **Quantum Core & Crystals:** Includes a rotating neon matrix wireframe sphere and floating interactive crystal nodes rendered in real-time.
* **Dual-Tone Cyberpunk Lighting:** Utilizes dynamic point lights (cyan and pink) to achieve a high-tech, glowing aesthetic.
* **Fully Responsive UI:** Built with a transparent HTML text overlay using Tailwind CSS, making it completely responsive across devices.

---

## 🛠️ Tech Stack Used

* **HTML5 & CSS3** - For structure and baseline setup.
* **Tailwind CSS** - For rapid, responsive UI styling and futuristic typography gradients.
* **Three.js** - The core 3D engine used to render the scene, geometries, materials, and lighting via the browser's WebGL.
* **GSAP (GreenSock Animation Platform)** - Handled smooth 60fps animations.
* **GSAP ScrollTrigger** - Tied the 3D camera's coordinates smoothly to the user's scroll behavior.
